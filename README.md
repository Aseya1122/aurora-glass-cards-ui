# Aurora Glass Cards UI

A modern and interactive card UI built using HTML, CSS, and JavaScript.

## Features
- Glassmorphism design
- Glowing gradient borders
- 3D tilt interaction (mouse-based)
- Smooth hover animations
- Fully responsive layout

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Effects)
- JavaScript (DOM + Mouse Interaction)

## Purpose
This project was built to practice modern UI techniques and advanced CSS effects.
